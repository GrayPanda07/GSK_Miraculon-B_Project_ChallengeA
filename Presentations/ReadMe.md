# Miraculon-B Clinical Trial Analysis

## Overview

This repository contains three presentations analysing the Miraculon-B clinical trial using different analytical tools and programming languages. The project focuses on identifying which patients are most likely to benefit from treatment and exploring factors that influence treatment response. The analysis was completed as part of a healthcare and data analytics portfolio project.

The presentations investigate:

* Treatment effectiveness compared to the control group
* Patient response patterns
* The impact of age, BMI, and protein concentration
* Predictive biomarkers for treatment response
* Data-driven insights to support clinical decision-making

## Key Findings

* Miraculon-B outperformed the control treatment group.
* Lower protein concentration was associated with better treatment response.
* BMI showed a moderate relationship with treatment response.
* Age showed minimal influence on patient outcomes.
* Protein concentration may serve as a predictive biomarker.
