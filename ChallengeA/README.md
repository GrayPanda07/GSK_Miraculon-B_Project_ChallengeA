# README: GSK "Step Up" Career Challenge - Path A

## Overview

This repository contains the resources for Path A of the GSK "Step Up" Career Challenge. The focus is on analysing clinical trial data to assess the effectiveness of Miraculon-B, a new cancer treatment, compared to the standard of care.

---

## Objective

Analyse clinical trial data to:
1. Evaluate the effectiveness of Miraculon-B.
2. Identify patient sub-groups that benefit most based on available data.

---

## Repository Contents

- **Data Files**:
  - `clinical-study.csv`: Patient demographics and treatment response data.
  - `protein-levels.csv`: Protein concentration data for predictive biomarker analysis.

- **Guides and Tools**:
  - `how-to-guide.pdf`: Full challenge details.
  - `excel_cheatsheet.pptx`: Excel tips for data analysis.
  - `analytical_roles.pdf`: Insights into pharmaceutical analytical roles.

-**Language files**
  -'GSK_challengeA_python.py': Python Code for the challenge.
  -'GSK_challengeA_R.r': R-Language code for the challenge.
  -'GSK_challengeA_Excel.xl': Excel/Google sheets code for the challenge.
  
- **Presentation Template**:
  - `Miraculon-B Clinical Trial_python.pptx`: For presenting findings from python code.
 - `Miraculon-B Clinical Trial_Excel.pptx`: For presenting findings from excel code.
 - - `Miraculon-B Clinical Trial_R.pptx`: For presenting findings from R-Language.
