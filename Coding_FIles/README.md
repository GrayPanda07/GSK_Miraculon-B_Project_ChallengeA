# Miraculon-B Clinical Trial Analysis

## Overview
This repository contains Python, R, and Excel-based analyses of the Miraculon-B clinical trial dataset. The project investigates treatment effectiveness and explores which patient characteristics are associated with a positive treatment response.

The analysis focuses on:
- Comparing Miraculon-B against the control treatment
- Investigating treatment response rates
- Exploring the impact of age, BMI, and protein concentration
- Identifying potential predictive biomarkers
- Producing visualisations and summary statistics
